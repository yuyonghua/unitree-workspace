# 硬件规格参考

## Unitree Go2-edu 硬件配置

### 主机

| 参数 | 规格 |
|------|------|
| 型号 | NVIDIA Jetson Orin NX 8GB |
| CPU | 6-core ARM Cortex-A78AE @ 2GHz |
| GPU | NVIDIA Ampere (1024 CUDA cores) @ 1GHz |
| RAM | 8GB LPDDR5 |
| 存储 | 64GB eMMC |
| 网络 | 1x Gigabit Ethernet |
| 操作系统 | Ubuntu 20.04 (JetPack 5.x) |

### 关节配置

| 参数 | 规格 |
|------|------|

| 关节电机 | Unitree 自研 servomotor |
| 关节范围 | 各关节不同, 参考官方文档 |
| 控制频率 | 500Hz |
| 通信协议 | CAN FD |

### IMU

| 参数 | 规格 |
|------|------|
| 型号 | 胸部内置 IMU |
| 陀螺仪 | 3轴 |
| 加速度计 | 3轴 |
| 采样频率 | 1000Hz |

## 激光雷达规格对比

### Livox LIS-3D (Go2-edu 标配)

| 参数 | 规格 |
|------|------|
| 水平视场角 (HFOV) | 70° |
| 垂直视场角 (VFOV) | 57° |
| 探测距离 | 0.1 - 40m |
| 点频 | 24,000 pts/s |
| 角分辨率 | 0.16° (水平) |
| 精度 | ±2cm @ 10m |
| 接口 | Ethernet |
| 功耗 | 8W |
| 重量 | ~260g |

### Livox Mid-360 (可选升级)

| 参数 | 规格 |
|------|------|
| 水平视场角 (HFOV) | **360°** |
| 垂直视场角 (VFOV) | 59° |
| 探测距离 | 0.1 - 40m |
| 点频 | 100,000 pts/s |
| 角分辨率 | 0.18° (水平) |
| 精度 | ±2cm @ 10m |
| 接口 | Ethernet |
| 功耗 | 12W |
| 重量 | ~440g |

### 对比总结

| 特性 | LIS-3D (标配) | Mid-360 (升级) |
|------|--------------|----------------|
| FOV | 70° 前向 | 360° 全向 |
| 适合场景 | 有前方固定视野 | 全向感知 |
| 建图漂移 | 较大 (转弯时) | 较小 (全向) |
| 价格 | 已包含 | ~1500-2000元 |
| 建议 | 室内小范围够用 | 建图 + 巡检推荐 |

## 功耗与续航

| 运行状态 | 功耗 | 续航 (电池 8000mAh) |
|---------|------|---------------------|
| 待机 | ~30W | ~2小时 |
| 站立 | ~50W | ~1.5小时 |
| 行走 (0.5m/s) | ~80W | ~50分钟 |
| 行走 (满速 4m/s) | ~150W | ~25分钟 |
| 建图 + 导航 | ~100W | ~40分钟 |
