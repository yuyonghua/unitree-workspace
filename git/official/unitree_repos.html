核心 SDK

⚙️
unitree_sdk2
C++ 核心开发套件 · v2.0.0
Unitree 机器人官方 C++ SDK，基于 CycloneDDS 实现高性能实时通信。提供统一编程接口控制全系四足和人形机器人，支持底层电机控制（500Hz）和高层运动行为调用。
Go2 Go2W B2 B2W A2 H1 G1 R1
  • DDS 发布/订阅通信，支持 500Hz 实时控制
  • RPC 客户端/服务端框架，30+ 高层运动行为
  • ROS 2 消息互操作（Twist, Odometry, PointCloud2 等）
  • 预编译静态库，支持 aarch64 和 x86_64
  • CRC32 完整性校验，Lease 安全访问控制
C++17 CycloneDDS CMake aarch64 / x86_64
🔗 GitHub 仓库
🐍
unitree_sdk2_python
Python SDK · v1.0.1
Unitree 官方 Python SDK，与 C++ 版本 API 对齐。提供高层运动控制、底层电机控制和 DDS 通信接口，适合快速原型开发和脚本化控制。
Go2 Go2W B2 B2W H1 H1_2 G1
  • Go2 运动客户端：30+ 行为（站立、行走、跳舞、翻跟头等）
  • G1 手臂动作：16+ 预定义手势（握手、拥抱、比心等）
  • G1 语音/TTS：语音合成、语音识别、音量/LED 控制
  • 底层 PD 电机控制，RecurrentThread 高精度定时
  • 视频流捕获（OpenCV）、游戏手柄支持（pygame）
Python ≥3.8 cyclonedds 0.10.2 numpy opencv

ROS 集成与 SLAM

🤖
unitree_ros2
ROS 2 集成包 · v0.3.0
Unitree 官方 ROS 2 集成方案。由于 SDK2 和 ROS 2 共用 CycloneDDS 中间件，可直接用原生 ROS 2 消息与机器人通信，无需额外 SDK 封装层。
Go2 B2 B2W H1 H1-2 G1
  • 3 个消息包：unitree_api、unitree_go (24种)、unitree_hg (11种)
  • 18+ 示例程序：底层控制、高层运动、手臂/灵巧手/语音
  • SportClient 封装 30+ 运动 API（站立、行走、翻转等）
  • 支持 ROS 2 Foxy (Ubuntu 20.04) 和 Humble (Ubuntu 22.04)
  • Docker 开发环境 + GitHub Actions CI/CD
C++14/20 ROS 2 CycloneDDS ament_cmake
🔗 GitHub 仓库
🗺️
point_lio_unilidar
3D SLAM 建图与定位
适配 Point-LIO 算法用于 Unitree L1/L2 激光雷达，实现高精度实时 3D SLAM。支持逐点状态估计，在剧烈运动下仍能保持鲁棒的定位与建图能力。
Unitree LiDAR L1 Unitree LiDAR L2
  • 逐点 ESEKF 状态估计，支持剧烈运动下的鲁棒定位
  • 增量 k-d 树管理稠密 3D 点云地图
  • IMU 融合运动畸变补偿
  • 自动保存 PCD 点云地图，RViz 实时可视化
  • 支持 5+ 种激光雷达类型（AVIA, Velodyne, Ouster, Hesai, Unitree）
C++14 ROS Noetic PCL Eigen3 OpenMP

仿真与强化学习

🔬
unitree_mujoco
MuJoCo 物理仿真器
基于 MuJoCo 的 Unitree 机器人仿真平台，实现"代码一次编写，仿真和真机均可运行"的 sim-to-real 开发流程。支持 7 种机器人模型，包含地形生成工具。
Go2 Go2W B2 B2W H1 H1-2 G1
  • C++ 和 Python 双版本仿真器
  • DDS 桥接：仿真与真机使用相同通信接口，仅切换 domain_id
  • 游戏手柄模拟（Xbox / Switch）
  • 地形生成器：台阶、粗糙地面、Perlin 噪声高度场等
  • 人形机器人虚拟弹性带辅助站立
C++17 / Python MuJoCo 3.3.6 CycloneDDS GLFW / OpenGL
🔗 GitHub 仓库
🧠
unitree_rl_gym
强化学习训练与部署框架
完整的四阶段 sim-to-real 强化学习流水线：Isaac Gym 训练 → 仿真验证 → MuJoCo 跨仿真测试 → 真机部署。使用 PPO 算法并行 4096 个环境高效训练。
Go2 H1 H1_2 G1
  • 4096 并行环境 GPU 加速训练（NVIDIA Isaac Gym）
  • LSTM 循环神经网络策略（G1/H1_2）
  • 步态相位感知 + 非对称 Actor-Critic
  • 域随机化：摩擦系数、质量扰动、周期性推力
  • TorchScript JIT 模型导出，C++/Python 真机部署
Python / C++ PyTorch Isaac Gym rsl_rl (PPO) MuJoCo 3.2.3
